About Full Stack Conf

The beautiful city of Portland, Oregon will be the host city for Full Stack Conf!

Explore the future of JavaScript with a lineup of industry professionals. Discover new techniques to advance your career as a web developer.

Expert Speakers

Our expert speaker lineup was just announced, so don't wait too long before grabbing your tickets!

Want to meet the international JavaScript community and share skills with some of the world's top experts, hackers, and makers? Be the first to know what to expect for the future of JavaScript.

Full Stack Conf is committed to being inclusive and welcoming for everyone. We look forward to another intensive day of learning and sharing.

Schedule

  • Keynote: Internet of Things
    9:00am

    NodeStradamus

    JavaScript is everywhere, on the browser, the server, and now on hardware. Finally–the Internet of Things is upon us — and it is powered by JavaScript. Even if you consider yourself a web developer, and you have no interest in “maker” culture, keep your eye on this space.

    Andrew Chalkley is a hardware hobbyist, software engineer and a full-time teacher at Treehouse.

  • React Basics
    10:00am

    Vivianne McVue

    We will build a small game during this tutorial. You might be tempted to skip it because you’re not building games — but give it a chance. The techniques you’ll learn in the tutorial are fundamental to building any React apps, and mastering it will give you a deep understanding of React.

  • Hey, Mongo!
    11:00am

    Jay Query

    The MongoDB Manual, including: CRUD, aggregation, database commands, mongo shell methods, and other common operations in MongoDB.

    MongoDB Stitch is a backend as a service that provides an HTTP API to MongoDB, integration with other services, and a declarative rules infrastructure which spans database and service actions.

  • Lunch
    12:00am

    Free pizza for everyone!

  • Introducing ES2015
    1:00pm

    Ecma Scriptnstuff

    Learn the basics of the ECMAScript 2015, also known as ES6, JavaScript standard. JavaScript has come a long way since it’s inception in 1995. The ECMAScript 2015 specification brought a lot of improvements, the largest set of changes the language has seen to-date. In this course, we’ll review some of the new features and demonstrate how they can be used to build better JavaScript applications.

  • Getting Started With Redux
    2:00pm

    Robbie Redux

    Redux provides a solid, stable and mature solution to managing state in your React application. Through a handful of small, useful patterns, Redux can transform your application from a total mess of confusing and scattered state, into a delightfully organized, easy to understand modern JavaScript powerhouse.

  • What's Babel?
    3:00pm

    Json Babel

    We're currently just a small group of volunteers that spend their free time maintaining this project. If Babel has benefited you in your work, becoming a contributor might be a great way to give back.

    Learn more about Babel by reading the get started guide or watching talks about the concepts behind it.
